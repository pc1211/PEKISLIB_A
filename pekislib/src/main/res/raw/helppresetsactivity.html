Set & Presets

Here you can edit a set of fields, forming a preset which can be added to a custom preset list.
Current preset field is displayed.
Press on current preset field name to select next field.

Without working with the custom preset list:
Select field to edit and set its value.
Press OK to accept current preset.

Working with the custom preset list:
You can add the current preset to the list.
Select (and deselect) a preset by touching it in the list.
A selected preset can be edited or removed from the list.
Press OK to accept current preset.

Button Function
(Current preset field name) > Select next preset field and its value
(Preset field value) Key in a new field value
(if a preset is selected in the list, it modifies it's corresponding field value)
(calls Input screen)
Add Add current preset to the list
Remove Remove preset from the list
(a preset must be selected first)
Default Reset current preset field to its default value
(if a preset is selected in the list, it modifies it's corresponding field value)
Cancel Cancel operation
(returns to previous screen)
OK Accept current preset
(returns to previous screen)
