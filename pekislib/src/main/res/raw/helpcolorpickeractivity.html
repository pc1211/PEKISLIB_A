Color screen

Here you can edit the colors displayed in the color wheel.
Current item's name is displayed and a small red pin points to its color.
Press on current item's name (or spin the wheel) to select next item.
Play with the three sliders (respectively R, G and B) to change its color or key in a 6 Hex chars value (RRGGBB).

Button Function
(Current item name) > Select next item and its color
RGB color value Key in a RGB color value
(calls Input screen)
Presets Set & Presets (Colors)
(calls Presets screen)
Cancel Cancel operation
(returns to previous screen)

(returns to previous screen)
